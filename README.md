**Order Microservice**
* **Objective**
  * A simple microservice that aims to apply the concepts of messaging and communication between microservices utilizing 
    Apache Kafka and Avro schemas.
  * It will receive a fake Order object by message, convert and persist on a DynamoDB database

* Stack
  * Java 17+
  * Spring Boot 2.7
  * Apache Kafka
  * DynamoDB
